WHERE

  • Used to filter a large number of results.
  • When used with text or date must be used with single quotes '.
  • When used with numbers no quotes should be used.

Equal

=

Used to compare values that are the same, numeric types only.

Equal

Example

SELECT * FROM atlantic
WHERE time = 600

Not Equal

<> or !=

Used to Exclude values which are not the same, numeric types only.

Not Equal

Example

SELECT * FROM atlantic
WHERE time <> 600
SELECT * FROM atlantic
WHERE time != 600

Greater Than

>

Used to select rows where the value in the named column is greater than the specified value

Greater Than

Example

SELECT * FROM atlantic
WHERE latitude > 28

Less Than

<

Used to select rows where the value in the named column is less than the specified value

Less Than

Example

SELECT * FROM atlantic
WHERE latitude < 28

Greater Than or Equal

>=

Used to select rows where the value in the named column is greater than or equal to the specified value

Greater Than or Equal

Example

SELECT * FROM atlantic
WHERE latitude >= 28

Less Than or Equal

<=

Used to select rows where the value in the named column is less than or equal to the specified value.

Less Than or Equal

Example

SELECT * FROM atlantic
WHERE latitude <= 28

Between

BETWEEN ... AND ...

Used to select rows where the value in the named column is BETWEEN the specified values.

Between

Example

SELECT * FROM atlantic
WHERE latitude BETWEEN 28 AND 30

In

IN (...)

Used to select rows where the value in the named column is one of those listed in brackets.

In

Example

SELECT * FROM atlantic
WHERE latitude IN (28, 29, 30)

Is

IS NULL

Used to select rows where the value in the named column is the special NULL value.

Is

Example

SELECT * FROM atlantic
WHERE Event IS NULL

Not

NOT

Used to invert any other command.

Not

Example

SELECT * FROM atlantic
WHERE Event IS NOT NULL
SELECT * FROM atlantic
WHERE latitude NOT IN (28, 29, 30)

Like

LIKE 'PATTERN'

Used to select rows where the value in the named column matches the pattern provided. Normally capitals and lower case do not matter.

Like

LIKE 'Harvey'

SELECT * FROM atlantic
WHERE name LIKE 'Harvey'

Matches when precisely 'HARVEY'.

Like

LIKE 'Har%'

SELECT * FROM atlantic
WHERE name LIKE 'Har%'

Matches when begins with 'HAR'.

Like

LIKE '%vey'

SELECT * FROM atlantic
WHERE name LIKE '%vey'

Matches when ends with 'VEY'.

Like

LIKE '%rve%'

SELECT * FROM atlantic
WHERE name LIKE '%rve%'

Matches when contains 'RVE' anywhere.

Like

LIKE '_ar%y'

SELECT * FROM atlantic
WHERE name LIKE '_ar%y'

Matches when 'ar' are the second and third letters and ends with a 'y'.

Like

LIKE 'H_____'

SELECT * FROM atlantic
WHERE name LIKE 'H_____'

Matches when the column begins with an 'H' and has precisely 6 letters.

And

AND

Used to compound two commands together where both must be satisfied.

And

Example

SELECT * FROM atlantic
WHERE Event IS NOT NULL AND Status LIKE 'HU'

Or

OR

Used to compound two commands together where either may be satisfied.

Or

Example

SELECT * FROM atlantic
WHERE Event IS NOT NULL OR Status LIKE 'HU'

Exercise 1

Show the names, dates and times of all the pacific rows that have a maximum wind speed of 80.

SELECT Name, Date, Time FROM pacific
WHERE [Maximum Wind] = 80

Exercise 2

Show all the columns of all the rows that represent atlantic hurricanes.

SELECT * FROM atlantic
WHERE Status LIKE 'HU'

Exercise 3

Show the latitude and longitude of all the atlantic rows that did not occur at midnight.

SELECT Longitude, Latitude FROM atlantic
WHERE TIME != 0

Exercise 4

Show the latitude and longitude of all the atlantic rows that were not a hurricane.

SELECT Latitude, Longitude FROM atlantic
WHERE Status NOT LIKE 'HU'

Exercise 5

Show the first 10 names, dates and times of all the atlantic rows that have a maximum wind speed of 80 and a time of midday

SELECT TOP 10 Name, Date, Time FROM atlantic
WHERE [Maximum Wind] = 80 AND Time = 1200

Exercise 6

Show the names, dates and times of all the atlantic rows that have a maximum wind speed between 75 and 125

SELECT Name, Date, Time FROM atlantic
WHERE [Maximum Wind] BETWEEN 75 AND 125

Exercise 7

Show the date of all the atlantic rows that have a latitude greater than 30 or a longitude less than -100

SELECT Date FROM atlantic
WHERE latitude > 30 OR longitude < -100

Exercise 8

Show the names of all Hurricanes whose names have exactly 3 letters in

SELECT Name FROM atlantic
WHERE Name LIKE '___' AND status LIKE 'HU'

Exercise 9

Show all the first 100 atlantic rows with all columns where the name contains an 'e' and an 'i'

SELECT Name, Date, Time FROM atlantic
WHERE Name LIKE '%e%i%' OR Name LIKE '%i%e%'

Exercise 10

Show all columns of all the atlantic rows that have a maximum wind speed of 80 or a latitude between 28 and 29 and a longitude between -98 and -99

SELECT * FROM atlantic
WHERE [Maximum Wind] = 80
OR (Latitude BETWEEN 28 AND 29 AND Longitude BETWEEN -99 AND -98)
